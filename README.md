# My-Notion Go Edition

一个独立的 React + Go 全栈学习项目，用于从零实现 My-Notion Web 端核心能力。

## 技术栈

- Web: React + Vite + TypeScript
- API: Go + Gin + GORM
- Database: PostgreSQL
- Monorepo: pnpm workspace + go work

## 快速开始

```bash
pnpm install
pnpm dev
```

Go API 单独启动：

```bash
pnpm dev:api
```

Web 单独启动：

```bash
pnpm dev:web
```
