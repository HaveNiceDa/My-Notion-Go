1. 接入成熟 i18n 库。
   - 为 `@my-notion-go/web` 安装 `i18next`。
   - 为 `@my-notion-go/web` 安装 `react-i18next`。
   - 原项目 Web 使用 `next-intl`，但 Go 版前端是 Vite SPA，因此本次采用更适合当前栈的 `i18next + react-i18next`。

2. 新增 i18n 基础设施。
   - 新增 `apps/web/src/i18n/resources.ts`。
   - 新增 `apps/web/src/i18n/index.ts`。
   - 支持 `zh` 和 `en` 两种语言。
   - 默认语言读取顺序：
     - 优先读取 `localStorage` 中的 `my-notion-go.language`。
     - 没有持久化语言时，根据 `navigator.language` 判断是否使用中文。
     - fallback 语言为中文。
   - 在 `apps/web/src/main.tsx` 中加载 i18n 初始化。

3. 新增语言切换组件。
