# 2026-05-16 10:31:02 shadcn UI 组件体系收口

## 背景

- 已完成 shadcn 基础设施接入后，项目仍有部分旧的手写按钮、输入框和弹窗样式。
- 本次目标是迁移 `AuthDialog` 和基础交互入口，避免后续继续堆叠旧 UI 写法。

## 实现

- `AuthDialog.tsx` 迁移到 shadcn 组件：
  - `Dialog`
  - `DialogContent`
  - `DialogHeader`
  - `DialogTitle`
  - `DialogDescription`
  - `Input`
  - `Button`
- 营销首页按钮迁移到 shadcn `Button`。
- 文档顶部导航按钮迁移到 shadcn `Button`。
- 空文档页新建按钮迁移到 shadcn `Button`。
- 工作区侧边栏折叠、新建、搜索、设置、主题切换和退出按钮迁移到 shadcn `Button`。
- 语言切换按钮迁移到 shadcn `Button`。

## 清理

- 删除不再使用的旧按钮样式：
  - `primary-button`
  - `ghost-button`
  - `icon-button`
  - `link-button`
- 删除不再使用的旧 Auth 弹窗和表单样式：
  - `auth-modal-backdrop`
  - `auth-modal`
  - `auth-modal-header`
  - `auth-form`
- 删除不再使用的旧 Dialog 样式：
  - `dialog-overlay`
  - `dialog-content`
  - `dialog-close`
  - `dialog-header`
  - `dialog-input`
  - `dialog-footer`
- 保留 `auth-modal-logo`，因为 Auth 弹窗和 LoadingScreen 仍复用品牌 logo 尺寸。
