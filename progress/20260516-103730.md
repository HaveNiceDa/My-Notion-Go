# 2026-05-16 10:37:30 BlockNote 国际化接入

## 背景

- 用户要求参考 BlockNote 官方 localization 文档，为编辑器接入国际化。
- 项目前端已使用 `react-i18next`，当前支持 `zh` 和 `en` 两种语言。

## 实现

- `DocumentEditor.tsx` 引入 `@blocknote/core/locales` 的 `zh` 和 `en` 字典。
- BlockNote 编辑器根据 `i18n.language` 选择字典：
  - `en` 使用 BlockNote 英文字典。
  - 其他情况默认使用 BlockNote 简体中文字典。
- 复用项目现有 `editor.placeholder` 文案，覆盖 BlockNote 字典里的：
  - `placeholders.default`
  - `placeholders.emptyDocument`
- `useCreateBlockNote` 通过 dependency list 监听字典变化，使语言切换后编辑器 UI 文案随之更新。

## 范围

- 本次只处理 BlockNote 自带 UI 文案，包括 slash menu、toolbar、side menu、table handle 等官方字典覆盖的内容。
- 项目自定义文案仍继续走 `apps/web/src/i18n/resources.ts`。

## 验证

- `pnpm --filter @my-notion-go/web typecheck` 通过。
- `pnpm --filter @my-notion-go/web build` 通过。
- Vite build 仍有大 chunk 提示，来自 BlockNote/editor 依赖，不影响本次国际化接入。
