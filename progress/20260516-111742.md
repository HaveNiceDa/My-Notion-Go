# 2026-05-16 11:17:42 前端生成规则固化与 AI Chat Tailwind 收口

## 背景

- 用户反馈 AI Chat 前端新增代码存在三类问题：
  - 注释不足，关键逻辑没有解释。
  - 仍有原生交互控件，没有完全使用 shadcn/ui 本地组件。
  - 新增了大量 feature 级全局 CSS，没有优先使用 Tailwind。

## 规则固化

- 已将以下规则写入项目 memory：
  - 新增前端组件样式优先使用 Tailwind utility class。
  - 避免新增大段 feature-specific global CSS。
  - 交互控件必须优先使用 `apps/web/src/components/ui` 下的 shadcn 风格本地组件。
  - 缺少 UI primitive 时先补本地组件封装。
  - 用户可见文案必须进入 `resources.ts` 的 `zh/en`。
  - feature hook 中生命周期清理和稳定回调优先使用 `ahooks`。
  - 新增代码需要对流式、缓存、鉴权、状态同步、并发、事务、边界处理等非显然逻辑写简洁注释。

## 实现

- 新增 `components/ui/textarea.tsx`，对齐 shadcn/ui 本地组件模式。
- `AIChatPanel.tsx` 调整：
  - 会话列表项使用 `Button`。
  - 输入框使用 `Textarea`。
  - 面板、header、会话列表、消息气泡、错误提示、输入区全部迁移为 Tailwind class。
  - 补充面板卸载、提交清空输入框等关键注释。
- `useAIChat.ts` 调整：
  - 使用 `useMemoizedFn` 和 `useUnmount`。
  - 补充 AbortController、React Query 与临时 streaming 消息同步、assistant 落库替换等注释。
- `global.css` 清理：
  - 删除 `.ai-chat-*` 大段 feature 样式。
  - 仅保留 `@keyframes blink` 作为可复用动画。
