## 设计决策

- AI Chat 继续使用 OpenAI Compatible `LLM_BASE_URL`；Embedding 使用 DashScope 原生多模态 API base URL。
- Embedding 优先读取 `DASHSCOPE_API_KEY`，未设置时 fallback 到 `LLM_API_KEY`。
- Qdrant 在 M5.0 阶段不是 API 强依赖；未启动 Qdrant 不应影响已有 MVP 功能。
- 启动阶段不真实调用 embedding API，避免消耗模型额度。
- Collection 名称仍走环境变量；embedding 模型和向量维度固定在代码中，模型切换必须经过代码审查和 collection 重建计划。
