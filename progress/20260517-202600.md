# 2026-05-17 20:26 M6.3 Public Documents

## 背景

- M6.0 搜索、M6.1 回收站、M6.2 收藏已完成基础闭环。
- 本次补齐 Notion-like 的发布页面能力。

## 本次改动

- 新增 `documents.public_id`：
  - 使用 UUID 作为公开链接 ID。
  - 不直接暴露内部 `documents.id`。
- 新增发布状态接口：
  - `POST /api/v1/documents/:id/publish`
  - `DELETE /api/v1/documents/:id/publish`
- 新增匿名公开只读接口：
  - `GET /api/v1/public/documents/:publicId`
  - 仅返回已发布、未归档、未删除文档。
  - 仅返回公开页面渲染所需字段。
- 前端顶部栏新增发布弹窗：
  - 支持发布 / 取消发布。
  - 支持复制公开链接。
- 前端新增公开页面：
  - 路由：`/p/:publicId`
  - 使用 BlockNote 只读渲染正文。
- `smoke-documents-api.mjs` 新增发布链路验证：
  - 发布文档并校验 `publicId`。
  - 匿名读取公开文档并校验正文。
  - 取消发布后校验公开接口返回不可访问。
  - 校验公开 DTO 不暴露内部 `id`、`path` 和 RAG 状态。

## 安全边界

- 公开接口不依赖登录态，但必须满足 `is_published = TRUE`。
- 公开接口过滤 `is_archived = FALSE` 和 `deleted_at IS NULL`。
- 公开 DTO 不返回 `user_id`、`path`、`isInKnowledgeBase`、排序字段等内部状态。
- 当前只发布单篇文档，不递归公开子页面。
